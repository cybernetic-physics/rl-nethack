# RL-NetHack Consolidated Research Docs

This folder consolidates the committed markdown trail into a smaller, durable set of documents. The goal is to preserve the reasoning, experiment history, implementation changes, blockers, and evaluation methodology without forcing future readers to reconstruct the project from dozens of point-in-time notes.

The docs are organized as:

- [01-project-goals-and-method.md](/home/luc/rl-nethack-worktree-20260416/docs/consolidated-2026-04/01-project-goals-and-method.md): the project thesis, what the repo is trying to accomplish, and the methodology that survived the iteration loop.
- [02-system-architecture.md](/home/luc/rl-nethack-worktree-20260416/docs/consolidated-2026-04/02-system-architecture.md): the main code paths, how the stack evolved, and what is actually trained today.
- [03-experimental-timeline.md](/home/luc/rl-nethack-worktree-20260416/docs/consolidated-2026-04/03-experimental-timeline.md): the chronological narrative of major experiments, code changes, and the conclusions that held up.
- [04-evaluation-and-benchmarks.md](/home/luc/rl-nethack-worktree-20260416/docs/consolidated-2026-04/04-evaluation-and-benchmarks.md): the tests, metrics, benchmark lineage, and why deterministic trace evaluation became the main gate.
- [05-blockers-and-next-steps.md](/home/luc/rl-nethack-worktree-20260416/docs/consolidated-2026-04/05-blockers-and-next-steps.md): the current bottlenecks, open questions, and recommended next steps.
- [06-literature-review-and-sota.md](/home/luc/rl-nethack-worktree-20260416/docs/consolidated-2026-04/06-literature-review-and-sota.md): the repo’s research notes, SOTA references, and the external literature that most strongly shaped design decisions.
- [07-operator-quickstart.md](/home/luc/rl-nethack-worktree-20260416/docs/consolidated-2026-04/07-operator-quickstart.md): the shortest practical guide to the current trusted loop, common commands, and benchmark hygiene.
- [99-source-index.md](/home/luc/rl-nethack-worktree-20260416/docs/consolidated-2026-04/99-source-index.md): a source map back to the original committed markdown files grouped by topic.

This consolidation is a synthesis, not a replacement of the originals. The new docs intentionally cite the earlier files so the reasoning trail remains auditable.
